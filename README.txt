# Vegetable Sales Analysis Project

This project performs data analysis on vegetable sales data using **Pandas** and **Matplotlib** in Python. The dataset contains information about vegetables, sellers, quantities, prices, and locations. The analysis provides insights into the total quantity of vegetables per location, the average price of each vegetable, and identifies which vegetable has the highest quantity.

## Installation

To run this project, you need to have **Python** installed on your machine along with the following libraries:

- **Pandas**: For data manipulation and analysis.
- **Matplotlib**: For creating visualizations.

To install the required libraries, you can use `pip`:
